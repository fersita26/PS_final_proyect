API Documentation

Base URL

http://127.0.0.1:5000

Endpoints

1. Listar Tareas

URL: /api/tasks

Método: GET

Autenticación: Requiere inicio de sesión.

Respuesta Exitosa:

[
    {
        "id": 1,
        "content": "Complete project",
        "deadline": "2024-12-15"
    },
    {
        "id": 2,
        "content": "Prepare presentation",
        "deadline": null
    }
]

2. Agregar Tarea

URL: /api/tasks

Método: POST

Cuerpo de la Solicitud:

{
    "content": "New Task",
    "deadline": "2024-12-20"
}

Respuesta Exitosa:

{
    "id": 3,
    "content": "New Task",
    "deadline": "2024-12-20"
}

3. Actualizar Tarea

URL: /api/tasks/<task_id>

Método: PUT

Cuerpo de la Solicitud:

{
    "content": "Updated Task",
    "deadline": "2024-12-25"
}

Respuesta Exitosa:

{
    "id": 3,
    "content": "Updated Task",
    "deadline": "2024-12-25"
}

4. Eliminar Tarea

URL: /api/tasks/<task_id>

Método: DELETE

Respuesta Exitosa:

{
    "message": "Task deleted successfully"
}
